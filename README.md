# Advanced Time Series Forecasting with Deep Learning and Attention

## Objective
Implement and evaluate attention-based deep learning models for multivariate
time series forecasting.

## Models
- Baseline LSTM
- LSTM with Additive Attention

## Metrics
- RMSE
- MAE
- SMAPE

## How to Run
1. python data_generation.py
2. python train.py

## Reproducibility
Random seeds are fixed for consistent results.
